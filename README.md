# Car Prediction Model

This project uses Linear Regression to predict car prices based on training data.

## Files

- `Linear-Regression-_Training-Data_-.csv` - Training dataset
- `Linear-Regression-Python-_Test-data_.csv` - Test dataset
- `Car Prediction Model.pdf` - Project documentation and analysis

## Project Overview

The model is trained using historical car data and predicts car prices using a Linear Regression algorithm.

## Requirements

- Python 3.x
- pandas
- numpy
- scikit-learn
- matplotlib (optional)

## Usage

1. Load the training dataset.
2. Train the Linear Regression model.
3. Load the test dataset.
4. Generate predictions.
